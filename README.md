# Portfolio Website

This is a simple portfolio website built using HTML, CSS, and JavaScript. It includes a responsive navigation bar, a profile image, and a welcome section.

## Features
- **Gradient Background**: A colorful gradient background for an eye-catching design.
- **Responsive Navbar**: A fixed navigation bar with links to different sections.
- **Profile Section**: Displays the user's name and image.

## File Structure
```
portfolio/
│-- index.html          # Main HTML file
│-- styles.css          # CSS for styling (if using an external file)
│-- script.js           # JavaScript for animations (if applicable)
│-- pages/
│   ├── qualifications.html
│   ├── skills.html
│   ├── projects.html
│   ├── contact.html
│-- images/
│   ├── photo.jpg       # Profile picture
│-- README.md           # Project documentation
```
## How to Use
1. Clone this repository:
   ```sh
   git clone https://github.com/Ajayvarma-github/portfolio.git
   ```
2. Navigate to the project folder:
   ```sh
   cd portfolio
   ```
3. Open `index.html` in your browser.
## Clone the repository
To host this portfolio on GitHub Pages:
1. Go to your repository settings.
2. Scroll down to the **GitHub Pages** section.
3. Set the source to **main branch**.
4. Site will be available at `https://Ajayvarma-github.github.io/portfolio/`.

## Customization
- Update `photo.jpg` with your profile picture.
- Modify `index.html` to include more personal details.
- Edit `styles.css` to change the appearance.
- Enhance `script.js` for more interactive effects.
